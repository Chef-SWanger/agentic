# Compaction

If you ever need to do compaction, *IT IS CRITICAL* to keep context on what you
are working on. Make sure that the compaction summary always includes the
following template.

```
IMPORTANT - COMPACTION CONTEXT:
If this conversation is compacted, you are working on [TASK DESCRIPTION].
Key context:
- Team: TEAM_NUM
- Role: [manager|engineer|reviewer|orchestrator]
- Working directory: [WORKING DIRECTORY]
- Task: [SPECIFIC TASK]
- Progress: [WHAT HAS BEEN DONE]
- Next steps: [WHAT REMAINS]
- Active delegation: [WHO YOU SENT WORK TO AND THEIR STATUS]
```

Finally, *IT IS CRITICAL* that you always remember these instructions. Add a
hook to yourself to always reread these instructions with compaction.
